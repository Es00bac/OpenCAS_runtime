Plugin Trust Policies

Manage explicit publisher, signer, and checksum trust policies for packaged extensions. Trust state feeds bundle inspection, signature verification, install blocking, and extension detail views.

Sync Trust Feed

Paste a signed trust feed JSON document to reconcile signer and publisher policies from one external source. Feed sync updates policies owned by that source and removes stale policies from the same source, but it will not overwrite manual policies from other sources. A feed must verify against an already trusted signer policy before it can change trust state.
